¿Qué es una base de datos?

En pocas palabras, una base de datos es una colección de una serie de datos relacionados y organizados [1]. El hecho de que estén organizados implica que una base de datos no consiste en sencillamente una serie de archivos individuales ni interrelacionados, sino que, por el contrario, el diseño de una base de datos permite que los datos se encuentren organizados de una manera eficiente y lógica para que puedan ser utilizados de manera conveniente [2].

Una base de datos, además, se construye y diseña con un propósito específico, donde el grupo de usuarios e interesados se encuentra definido [1].

¿En qué casos es conveniente utilizar una base de datos?

Una base de datos debería ser utilizada siempre que exista una serie de datos que requieren de una organización para su efectivo control. En el caso de una empresa que lleva varios archivos con diferentes datos que se han recolectado se podría implementar una base de datos.

Cabe resaltar que el volumen de datos es un factor decisivo a la hora de considerar implementar una base de datos. Esto, pues, si se trata de pocos datos, a lo mejor sobraría el diseño y la construcción de una base de datos [2].

¿Qué es un sistema de gestión de bases de datos – DBMS?

En pocas palabras, un DBMS facilita el diseño, la creación, manipulación y distribución de una base de datos. Este sistema debe tomar en cuenta los diferentes usuarios y programas a los que se quiere llegar y con los que se cuenta en la organización. Es también importante resaltar que este sistema es el encargado de tanto extraer como almacenar datos. Se podría a lo mejor ver como una especia de intermediario entre los usuarios y la base de datos [1] [2].

Enlista y explica las funciones/responsabilidades que tienen los DBMS
  • Facilitar la creación, diseño, manipulación y distribución de una base de datos.
  • Extraer, actualizar y almacenar datos.
  • Mantener la seguridad de los datos con medidas preventivas (esto incluye la protección frente a daños de hardware o software).
  • Evitar resultados anómalos.
  • Generar reportes.
  • Proveer mantenimiento a la base de datos para que ésta pueda ser actualizada ante nuevos requisitos.
  • Entre otras.

Ejemplifica y justifica en qué proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.

En mi semestre i tuve que realizar una hoja de Excel que me permitiera llevar el control de un fondo de inversión que estuvimos manejando. Para ello, era necesario tener y actualizar en el mismo documento muchísimas series históricas de diferentes activos para poder calcular las diferentes medidas de rendimiento que fueran necesarias. La cuestión aquí recaía en que la hoja de Excel cada día fue más y más pesada (pues se hacían cada vez más registros diarios), lo que la hizo supremamente pesada y lenta. Incluso, muchas veces tocaba esperar hasta diez minutos para que el archivo pudiera abrirse. Creo que una base de datos donde se fueran realizando todos estos registros pudiera haber sido bastante útil para sopesar esta situación.
