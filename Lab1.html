Descripcion de Conceptos

  • URL (Uniform Resource Locator): Es un indentificador de recurso uniforme, es la direccion unica que tienen todos los elementos que estan en internet, desde fotos, videos, audios entre otros. [1]
  • MIME (Multipurpose Internet Mail Extensions): Es un estandar que permite a los usuarios insertar archivos en los correos, defininiendo la naturaleza y el formato del documento. [2]
  • HHTP(Hypertext Transfer protocol): Es un protocolo para la transferencia de informacion o conectarse a servidores web que estan en el internet y asi establecer conexión con estos servicios.[3]
    • Error 200 - Todo esta bien
    • Error 304 - No modificado
    • Error 401 - Acceso no permitido
    • Error 404 - No encontrado
    • Error 500 - Error del servidor
  • XML(Extensible Markup Language): Es usada en la descripcion de la informacion. Es una manera mas flexible de poder crear formatos de informacion y que mas adelante pueden ser compartidas electronicamente por medio del internet o de redes.[4]
  • XHTML (Extensible HyperText Markup Language): Es un lenguaje de marcado de texto que fue desarrollada con las fortalezas de XML y HTML. Es muy similar a HTML pero algo mas robusto y se aconseja normalmente para la momdelacionde paginas web.[5]
  • HTML5 (Hypertext Markup Language revision 5): Es un lenguaje de marcado para el contenido de la web en el que se desarrolla la estructura y la presentacion. [6]
  • JavaScript: Es un lenguaje de script que es utilizada mayormente en la Web. Es mas que todo utilizada para mejorar el codigo de HTML. Es un lenguaje interpretado por lo que no tiene que ser compilado.[7]
  • PHP (Hypertext Preprocessor): Es un lenguaje de scripting para crear contenido dinamico o interactivo para paginas web HTML. El servidor procesa los comandos de PHP y le envia los resultados a el navegador del visitante.[8]
  • JSP (JavaServer Pages): Es una tecnologia que es usada para desarrollar paginas web interactivas. Es una mejor version de Java servlets y separa la logica de negocio con la capa de presentacion.[9]
  • AJAX:): Es una tecnica de desarrollo que es usada para crear una aplicación web interactiva. Combina funciones como XHTML y CSS para la presentacion, XML y XSLT para el intercambio de datos y capturacion de informacion asincronica con XML HTTP Request. Todo esto es juntado con JavaScript.[10]

Preguntas

Cuál es la diferencia entre Internet y la World Wide Web?

El internet se refiere a la infraestructura de interconexiones físicas y digitales, conectando los dispositivos a una sola red. Mientras que la World Wide Web es la forma o el medio en el que se accede a la información del internet. [11]

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

El propósito de estos métodos es manejar toda la comunicación en la web.

  • GET: Recuperar datos
  • HEAD: Recupera o manda información del encabezado
  • POST: Manda información al servidor de manera más segura para hacer el manejo de la información.
  • PUT: Actualiza la información del servidor
  • PATCH: Actualiza la información parcial del recurso o servidor especifico
  • DELETE: Eliminar un registro

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

Antes que todo el estándar HTML 4 ya está establecido mientras que el HTML 5 sigue estando en constante actualización. Por una parte, la sintaxis esta más simplificada, para la declaración del tipo de documento son ahora más simple con DOCTYPE HTML!. Otra de las adiciones en HTML5 son el cambio de Flash con el elemento canvas y su utilidad para representar gráficos. Los nuevos elementos header y footer para el especifico de estas dos partes de la página web. Hay muchas otras adiciones en HTML5, pero estas son las principales diferencias con HTML4. [13]
