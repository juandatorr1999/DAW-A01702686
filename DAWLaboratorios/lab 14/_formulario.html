Preguntas:

¿Qué es ODBC y para qué es útil?

ODBC es un estándar para poder acceder a una base de datos desde cualquier aplicación independientemente del gestor que esta BD tenga.

¿Qué es SQL Injection?

Consiste en insertar código SQL en los campos de algún formulario, pudiendo alterar así, la base de datos.

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Para evitarlo, las entradas deben ser limpiadas por el lado del servidor, tal y como se hace con el html.
