¿Qué diferencias y semejanzas hay entre Java y JavaScript?
Diferencias
  • JavaScript no sirve para crear aplicaciones que se sostienen por si mismas, mientras que Java sí (corren en máquinas virtuales).
  • JavaScript es comúnmente utilizado dentro de un HTML y le da interactividad al documento, diferente a Java cuya función no es para esto.
  • Los Plug-ins requeridos por cada uno de los lenguajes para su ejecución son diferentes.
  • La forma de compilación y de ejecución de los códigos es diferente, lo que hace que el debugging sea diferente en cada caso.
Similitudes
  • Tanto Java como JavaScript son lenguajes de programación.
  • Ambos lenguajes pueden ser ejecutados en exploradores. Sin embargo, las aplicaciones de Java han ido perdiendo popularidad por este medio.
  • Los dos lenguajes usan bibliotecas y frameworks que permiten reusar código.

¿Como se declara una variable con alcance local dentro de una función?

La variable se declara de igual forma que se declara cualquier otra variable, solo que en este caso como se crea dentro de la funcion lo que hace que solo este disponible dentro de la misma.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Las implicaciones, a lo mejor, podrían incluir que el desarrollador vuelva a utilizar variables globales que había declarado anteriormente, cambiando su valor a sin que él se de cuenta, lo que puede conducir a un error.
