Preguntas

¿Por qué es una buena práctica usar JavaScript para checar que sean válidos los inputs de las formas antes de enviar los datos al servidor?

Es buena idea para evitar que se almacene información con errores en el servidor, pues podría traer consecuencias negativas tanto para el usuario como para el administrador.

¿Cómo puedes saltarte la seguridad de validaciones hechas con JavaScript?

Para lograr esto, el usuario puede sencillamente deshabilitar o "apagar" javascript para evitar que éste le genere obstrucciones al querer enviar un formulario

Si te puedes saltar la seguridad de las validaciones de JavaScript, entonces ¿por qué la primera pregunta dice que es una buena práctica?

La validación con Javascript permite que ésta se haga de manera ágil y que sea amigable. Sin embargo, por motivos de seguridad, pueden haber problemas.
